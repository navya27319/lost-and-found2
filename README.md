# Lost and Found Portal

A simple web-based Lost and Found system where users can report lost and found items easily.

---

## 🚀 Features

- Report lost items
- Report found items
- Simple and clean UI
- Responsive design
- Easy navigation

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
- Vite

---

## ⚙️ Installation

```bash
git clone https://github.com/navya27319/lost-and-found.git

cd lost-and-found

npm install

npm run dev
```
